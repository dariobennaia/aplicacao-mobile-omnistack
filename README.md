# Aplicação Mobile (Omnistack - 10)

Esta é uma aplicação mobile, desenvolvida juntamente com o Diego da Rocketseat através de suas vídeo aulas

## Requisitos
- Node >= 12
- Expo

## Instalação

A instalação é bem simples basta rodar um dos comando abaixo.
```bash
yarn install
```
ou 
```bash
npm install
```
